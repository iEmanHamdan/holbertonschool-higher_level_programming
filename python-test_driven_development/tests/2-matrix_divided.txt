========================================
The ``2-matrix_divided`` Testing Module
========================================

This module provides test validation cases for the `matrix_divided` function.
The function divides all numerical elements of a matrix by a given value.

Using the Function
------------------
    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Basic Matrix Division Matrix:
    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

Confirming the Original Matrix Remained Intact:
    >>> matrix
    [[1, 2, 3], [4, 5, 6]]

Floating Point Division and Negative Handling:
    >>> matrix_divided([[10.5, -20.0]], 2.5)
    [[4.2, -8.0]]
